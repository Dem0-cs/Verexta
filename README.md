# Verexta 🚀
> A lightweight, speedy, and stable Discord selfbot written in Go.

Verexta is designed for efficiency and minimal resource usage, leveraging the speed of Go to provide a seamless experience without the bloat of traditional framework-heavy bots.

---

## ⚠️ Disclaimer
**Selfbotting is against Discord's Terms of Service.** Using this software can result in your account being permanently banned. The developers of Verexta are not responsible for any actions taken against your account. Use this tool for educational purposes only and at your own risk.

---

## ✨ Features
* **Blazing Fast:** Built with Go for near instant execution.
* **Low Footprint:** Minimal CPU and RAM usage.
* **Stable:** Stress tested to the max, Verexta is one of the most stable selfbots there is!
* **Open Source:** Free to audit, modify, and learn from.
